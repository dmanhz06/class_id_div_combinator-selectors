Class ID selector css

  1. ID selector:
    • Chọn phần tử riêng muốn thay đổi css mà nó bị trùng thẻ với các phần tử khác
      => Sử dụng ID selector (mỗi phần tử chỉ có 1 ID duy nhất ko được trùng nhau)
    • Cú pháp: #id_name{ thuộc tính ...}
    • NOTE: mỗi phần tử chỉ có 1 ID duy nhất => nếu muốn áp dụng cho nhiều phần tử riêng thì phải dùng class
  2. Class selector:
    • Chọn nhiều phần tử riêng muốn thay đổi css mà nó bị trùng thẻ với các phần tử khác
      => Sử dụng Class selector (mỗi phần tử có thể có nhiều class khác nhau)
    • Cú pháp: .class_name{ thuộc tính ...}
    • NOTE: mỗi phần tử có thể có nhiều class khác nhau

*NOTE: cách code nhanh với class:

ví dụ muốn gõ <h1 class="helo">hello</h1>

thì chỉ cần gõ : h1.helo + enter là ra được đoạn code trên

Combinator selector

  1. Code áp dụng cho đối tượng riêng có chung thẻ (h1, h2, p...)

    2. Ví dụ:

            
<h1>hello mọi người</h1>
<h1 class="kon">konichiwa mina </h1>

    => để phân biệt thì cho thêm id vào h1 rồi dùng cú pháp:

    h1.kon{ thuộc tính ...}

    => Áp dụng cho h1 có class="kon"

  2. Code áp dụng cho toàn đối tượng trong css:

    3. *{ thuộc tính ....}

    Ví dụ:

    *{
    color: red;
    font-size: 20px;
    }

    => Áp ụng cho toàn bộ đối tượng thành màu đỏ và cỡ chữ 20px

  3. Code áp dụng cho nhiều thẻ cùng lúc:

    4. Ví dụ:

    muốn thẻ h1,h2,h3,p,ul đều màu đỏ

    => ta dùng dấu phẩy đối vs các thẻ tag

    h1, h2, h3, p, ul{
    color: red;
    }

    Ngoài ra còn có thể dùng dấu phẩy để liệt kê đối vs các class và id

    Ví dụ:

    muốn h1, class="class2", id="id1" đều có nền xanh

    => ta dùng dấu phẩy đối vs các class

                        
h1,
.class2,
#id1{
    background-color: blue;
}

    *NOTE: Ngoài ra ta còn có thể sử dụng 2 class cùng lúc tượng trưng như dùng and

    Ví dụ:

    - muốn thẻ p có cả class="class1" và class="class2" thì mới áp dụng css

    => ta dùng dấu chấm nối tiếp 2 class

    p.class1.class2{
    color: red;
    }

    => Áp dụng cho thẻ p có cả class="class1" và class="class2"

sử dụng div

- Div là thẻ dùng để nhóm các phần tử lại với nhau để dễ dàng áp dụng css hoặc js

* div p: là chọn tất cả các thẻ p trong thẻ div
  • Ngoài ra còn có cả (div class) p : là chọn tất cả các thẻ p trong thẻ div class
  • ví dụ như hình sau:

    • div > p: là chọn tất cả các thẻ p là con trực tiếp của thẻ div

    div > p : p là con trực tiếp của div

    => nếu trong p còn có con mở rộng thì khi dùng div>p sẽ không ảnh hưởng đến con nằm trong p ( vì nó ko trực tiếp)
